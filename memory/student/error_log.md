# 错题本

> 每一道错题都是一次成长的机会。记录错误，才能避免重蹈覆辙。

## 记录格式

| 日期 | 题目 | 知识点 | 错误原因 | 状态 | 复习日期 |
|------|------|--------|---------|------|---------|
| | | | | 🟡待复习 | |

### 错误原因分类
- 🧠 思路错误：解题方向不对
- 📝 代码错误：语法 bug、边界情况漏了
- ⏰ 时间超限：算法复杂度太高
- 💾 空间超限：数组开太大
- 😵 完全不会：没有思路
- 🤦 粗心：看错题、漏条件

### 状态标记
- 🟡 待复习
- 🟢 已掌握
- 🔴 仍不会
