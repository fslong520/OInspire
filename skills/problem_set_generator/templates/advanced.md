# 提高阶段训练模板

## 适用对象
- 已有扎实基础
- 目标：NOIP 提高组 / CSP-S / 省选入门

## 训练主题分配

### Week 1-2: 图论基础
- BFS/DFS 进阶
- 最短路（Dijkstra、SPFA、Floyd）
- 拓扑排序
- 最小生成树

### Week 3-4: 动态规划
- 区间 DP
- 树形 DP
- 状态压缩 DP

### Week 5-6: 高级数据结构
- 并查集进阶
- 树状数组
- 线段树入门

### Week 7-8: 数学与字符串
- 组合数学基础
- 数论（gcd、lcm、快速幂、逆元）
- 字符串基础（Trie、KMP）

## 建议题量
- 每天 5-10 题
- 总题量约 200-250 题
- 通过率目标：50% 以上
- 每周至少 1 场模拟赛
