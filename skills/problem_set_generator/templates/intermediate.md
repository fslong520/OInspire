# 基础阶段训练模板

## 适用对象
- 已掌握基础语法
- 目标：CSP-J 提高 / NOIP 普及组 / 蓝桥杯省赛

## 训练主题分配

### Week 1-2: 排序与二分
- 快速排序、归并排序
- 二分查找、二分答案
- 离散化入门

### Week 3-4: 基础数据结构
- 栈、队列、单调栈/队列
- 哈希表、map/set
- 前缀和与差分

### Week 5-6: 搜索
- DFS 基础与剪枝
- BFS 基础
- 简单回溯

### Week 7-8: 贪心与简单 DP
- 经典贪心模型
- DP 入门（线性 DP）
- 背包问题（01、完全）

## 建议题量
- 每天 5-8 题
- 总题量约 150-200 题
- 通过率目标：60% 以上
