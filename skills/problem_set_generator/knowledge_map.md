# 信息学竞赛知识点地图

## 📊 知识体系概览

```
入门 ──→ 基础 ──→ 提高 ──→ 进阶 ──→ 冲刺
```

## 详细知识点

### 🟢 入门阶段

| 知识点 | 核心概念 | 常见题型 |
|---------|---------|---------|
| 变量与类型 | int, long long, double, char, bool | 计算、转换 |
| 运算符 | +, -, *, /, %, 位运算 | 简单计算 |
| 条件语句 | if, else if, else, 三目运算符 | 判断、分支 |
| 循环 | for, while, do-while, break, continue | 累加、枚举 |
| 函数 | 参数、返回值、递归基础 | 模块化编程 |
| 数组 | 一维数组、遍历、最值查找 | 统计、排序 |
| 字符串 | char 数组、string、基本操作 | 处理、匹配 |

### 🟡 基础阶段

| 知识点 | 核心概念 | 常见题型 |
|---------|---------|---------|
| 二维数组 | 矩阵操作、方向数组 | 迷宫、地图 |
| 结构体 | struct、排序 | 排序、分组 |
| 排序算法 | 冒泡、选择、插入、快排、归并 | 排序、第K大 |
| 栈与队列 | LIFO、FIFO、STL | 括号匹配、模拟 |
| 贪心 | 局部最优 | 区间覆盖、霍夫曼 |
| 二分 | 二分查找、二分答案 | 最小值最大化 |
| 前缀和 | 一维、二维 | 区间查询 |
| 差分 | 区间修改、单点查询 | 区间更新 |
| 简单搜索 | DFS、BFS | 连通块、最短路径 |
| 简单 DP | 线性 DP、状态定义 | 最长上升子序列 |

### 🟠 提高阶段

| 知识点 | 核心概念 | 常见题型 |
|---------|---------|---------|
| 图论基础 | 建图、遍历 | 最短路、连通性 |
| 最短路 | Dijkstra、SPFA、Floyd | 路径规划 |
| 最小生成树 | Kruskal、Prim | 最小代价 |
| 拓扑排序 | DAG、入度 | 依赖关系 |
| 背包 DP | 01、完全、多重 | 价值最大化 |
| 区间 DP | 区间划分、合并 | 石子合并 |
| 树形 DP | 树的遍历、子树 | 树的最大权值 |
| 并查集 | 路径压缩、按秩合并 | 连通性维护 |
| 树状数组 | lowbit、单点修改 | 区间查询 |
| 线段树 | 建树、懒标记 | 区间操作 |
| 组合数学 | C(n,k)、容斥 | 计数 |
| 数论 | gcd、lcm、快速幂、逆元 | 模运算 |

### 🔴 进阶阶段

| 知识点 | 核心概念 | 常见题型 |
|---------|---------|---------|
| 高级 DP | 状态压缩、数位 DP | 复杂状态 |
| 强连通分量 | Tarjan、缩点 | 有向图连通性 |
| 二分图 | 匹配、匈牙利 | 最大匹配 |
| AC 自动机 | Trie + 失配指针 | 多串匹配 |
| 后缀数组 | 高度数组 | 字符串统计 |
| 平衡树 | Treap、Splay | 动态维护 |
| 可持久化 | 主席树 | 历史版本查询 |

### ⚫ 冲刺阶段

| 知识点 | 核心概念 | 常见题型 |
|---------|---------|---------|
| 网络流 | 最大流、费用流 | 匹配、分配 |
| 后缀自动机 | SAM、状态转移 | 子串问题 |
| 计算几何 | 叉积、凸包 | 几何判断 |
| 博弈论 | SG 函数、Nim | 必胜必败 |
| 莫队算法 | 离线查询 | 区间统计 |
| CDQ 分治 | 三维偏序 | 高维查询 |
| 生成函数 | 多项式、FFT | 组合计数 |

## 前置关系（学习顺序参考）

```
基础语法 → 数组 → 排序 → 二分 → 前缀和/差分
                                         ↓
搜索(DFS/BFS) → 贪心 → 简单 DP → 背包 → 区间/树形 DP
    ↓
图论(建图) → 最短路 → MST → 拓扑
    ↓
数据结构: 并查集 → 树状数组 → 线段树 → 平衡树
    ↓
字符串: Trie → KMP → AC自动机 → 后缀数组/自动机
    ↓
数学: 组合 → 数论 → 博弈 → 生成函数
```
